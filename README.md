# AI Chatbot — Powered by Puter.js (100% Free)

No API key needed. Uses Puter.js to access free AI models.

## Setup

1. Install dependencies: `npm install`
2. Run: `npm run dev`
3. Open http://localhost:5173 — that's it!

## Free Models Available
- Llama 3.3 70B (Meta)
- DeepSeek R1
- Gemini 2.0 Flash
- Mistral Small 3.1
- Claude Haiku

## No .env.local needed!
Puter.js handles all authentication for free via their "User-Pays" model.
